Program Data Base debugging information format.
This namespace contains unopinionated types and data definitions only. For an implementation of parsing and caching PDB information, see `std.debug.Pdb[](https://ziglang.org/documentation/master/std/#std.debug.Pdb)`.
Most of this is based on information gathered from LLVM source code, documentation and/or contributors.
