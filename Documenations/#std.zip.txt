Decompresses the given data from `reader` into `writer`. Stops early if more than `uncompressed_size` bytes are processed and verifies that exactly that number of bytes are decompressed. Returns the CRC-32 of the uncompressed data. `writer` can be anything with a `writeAll(self: *Self, chunk: []const u8) anyerror!void` method. `pub fn extract[](https://ziglang.org/documentation/master/std/#std.zip.extract)(dest: std[](https://ziglang.org/documentation/master/std/#std).fs[](https://ziglang.org/documentation/master/std/#std.fs).Dir[](https://ziglang.org/documentation/master/std/#std.fs.Dir), seekable_stream: anytype, options: ExtractOptions[](https://ziglang.org/documentation/master/std/#std.zip.ExtractOptions)) !void`
    
Extract the zipped files inside `seekable_stream` to the given `dest` directory. Note that `seekable_stream` must be an instance of `std.io.SeekableStream[](https://ziglang.org/documentation/master/std/#std.io.seekable_stream.SeekableStream)` and its context must also have a `.reader()` method that returns an instance of `std.io.Reader[](https://ziglang.org/documentation/master/std/#std.io.GenericReader)`. `pub fn findEndRecord[](https://ziglang.org/documentation/master/std/#std.zip.findEndRecord)(seekable_stream: anytype, stream_len: u64) !EndRecord[](https://ziglang.org/documentation/master/std/#std.zip.EndRecord)`
    
Find and return the end record for the given seekable zip stream.
